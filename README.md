# basic-webboard-frontend
a simple webboard , that create for learning to use React .

<br>
<br>
# how to run
1.get basic-webboard-frontend, basic-webboard-backend, basic-webboard-docker repository in same folder<br>
2.use "npm i" in basic-webboard-frontend and basic-webboard-backend<br>
3.run "docker compose up -d " in root of 3 repository. (you must have docker , docker-compose)<br>
